# Schedulerjs

##### Scheduler used for CRON module
#

> The payload exemple received
#
```javascript
// Exemplo object receive
{
    target          : 22,
    stripMin        : 10,
    stripMax        : 25,
    day             : 5,
    enableDay       : 1,
    timeSwitchedOn  : '09:41',
    timeSwitchedOff : '16:00'
}
```
